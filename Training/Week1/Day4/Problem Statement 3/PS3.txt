Problem 3: Simple Discount Calculator (Level-1)
Scenario
An online store wants to apply a discount based on the
 total purchase amount.
📌 Requirements
•	Store purchase amount in a variable
•	Apply discount rules:
o	Amount ≥ 5000 → 20% discount
o	Amount ≥ 3000 → 10% discount
o	Amount < 3000 → No discount
•	Calculate and display:
o	Discount amount
o	Final payable amount

🛠️ Technical Constraints
•	Use arithmetic operators
•	Use if–else statements
•	Use only primitive data types
No user input (hardcoded values allowed)
🎯 Learning Outcome
You will be able to:
•	Perform calculations using operators
•	Work with expressions
•	Apply conditional statements
•	Build real-world logic using JavaScript basics
