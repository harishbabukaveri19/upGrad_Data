Level-2 Problem 3: College Event Management Dashboard

Scenario:
Design a responsive dashboard for managing college events including event list, add event form, and navigation.

📌 Requirements
- Use Bootstrap navbar with collapse functionality.
- Implement responsive grid layout for dashboard sections.
- Use form components for event registration.
- Implement modal for adding new events.
- Use Bootstrap utilities for alignment and spacing.

🛠️ Technical Constraints
- Use Bootstrap 5 CDN.
- No external UI frameworks.
- Proper breakpoint usage (sm, md, lg, xl).
- Modal must work using Bootstrap JS bundle.

Expectations:
- Dashboard layout adapts across devices.
- Modal opens and closes correctly.
- Form fields use proper Bootstrap validation styling.

🎯 Learning Outcome
- Implement advanced components like modals.
- Create structured responsive dashboards.
- Apply responsive behaviors effectively.
