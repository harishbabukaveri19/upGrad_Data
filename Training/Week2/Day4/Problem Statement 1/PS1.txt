Level-1 Problem 1: Responsive Personal Portfolio Homepage

Scenario:
You are designing a responsive personal portfolio homepage for a fresher web developer who wants to showcase profile information, skills, and contact details.

📌 Requirements
- Integrate Bootstrap 5 using CDN.
- Create a responsive layout using container, rows, and columns.
- Use Bootstrap grid system with appropriate breakpoints.
- Include components such as navbar, cards (for skills), and buttons.
- Use utility classes for spacing, alignment, and colors.

🛠️ Technical Constraints
- Must use Bootstrap 5 CDN (no local files).
- No custom CSS except minimal styling if required.
- Use grid system (row and col classes) properly.
- Ensure responsiveness across mobile, tablet, and desktop.


🎯 Learning Outcome
- Understand Bootstrap setup using CDN.
- Learn responsive grid system and breakpoints.
- Implement basic Bootstrap components.
- Apply utility classes effectively.
